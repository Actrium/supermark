# Footnote Feature

为 Supramark 提供脚注支持。

## 功能

- 脚注引用
- 脚注定义

## 使用

查看 examples 目录获取更多示例。

## API 参考

### 函数

#### `createFootnoteFeatureConfig`

创建 Footnote Feature 配置对象，用于在 SupramarkConfig 中启用脚注支持

#### `options`

Footnote Feature 配置选项（当前为空对象）

### 接口

#### `FootnoteFeatureOptions`

Footnote Feature 的配置选项接口（当前为空，保留用于未来扩展）
