# GFM Feature

为 Supramark 提供 GitHub Flavored Markdown 扩展支持。

## 功能

- 删除线
- 任务列表
- 表格

## 使用

查看 examples 目录获取更多示例。

## API 参考

### 函数

#### `createGFMFeatureConfig`

创建 GFM Feature 配置对象，用于在 SupramarkConfig 中启用 GitHub Flavored Markdown 扩展

#### `options`

GFM Feature 配置选项（当前为空对象）

### 接口

#### `GFMFeatureOptions`

GFM Feature 的配置选项接口（当前为空，保留用于未来扩展）
