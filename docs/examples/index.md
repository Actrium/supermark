# 示例项目

Supramark 提供完整的示例项目，展示在不同平台上的实际使用方法。

### [React Web CSR 示例](./react-web-csr)

完整的可运行项目，展示 Supramark 在该平台的使用方法。

### [React Native 示例](./react-native)

完整的可运行项目，展示 Supramark 在该平台的使用方法。

## 运行示例

所有示例项目都可以直接克隆并运行：

```bash
git clone https://github.com/supramark/supramark.git
cd supramark
bun install
cd examples/react-web
bun run dev
```
