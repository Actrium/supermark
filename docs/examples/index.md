## 运行示例

所有示例项目都可以直接克隆并运行：

```bash
# 克隆仓库
git clone https://github.com/supramark/supramark.git
cd supramark

# 安装依赖
npm install

# 运行示例（以 React Web 为例）
cd examples/react-web
npm run dev
```
